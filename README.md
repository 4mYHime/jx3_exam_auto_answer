# 这是什么
一个基于OCR识别的剑网三科举智能答题器

# 注意
我删除了代码中的app_id和app_key，如果您想试着运行的话，可以登陆腾讯AI开放平台(https://ai.qq.com/)进行申请。本应用使用的API为“计算机视觉”分类下的“通用OCR”

此项目不提供已编译的exe文件

另外，此项目暂时只支持windows系统

# 如何使用
·开始运行

·去考试的地方

·在答题器上设置你的窗口比例（游戏中ESC→界面设置 就能看到）

·单击“开始查题”按钮

·切回游戏 按一下F9

·等答案刷出来以后手动选择答案 然后单击下一题

·答完最后一题以后 按下F10退出
